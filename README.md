# E-COMMERCE-DATA-DASHBOARD


https://github.com/user-attachments/assets/e130d308-2a38-4198-999f-f2fa5acd40b2



### 🎯 Problem & Solution
Developed a performance-focused dashboard to track sales metrics and identify "Top Achievers" within the sales team.

### 🛠 Tools & Technologies
* **Power BI:** For building the interactive performance dashboard.
* **Power Query:** Used for data cleaning and transforming e-commerce datasets.
* **Advanced DAX:** Created complex measures to calculate individual and team performance metrics.

### 💡 Business Impact
Provided clear visibility into staff productivity, leading to a more data-driven incentive and reward system.
